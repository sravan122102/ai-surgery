# SurgiScore

SurgiScore is a Flask-based surgical training platform that analyzes arthroscopic procedure videos and produces objective skill scores, visual reports, and AI-generated coaching feedback. It combines computer vision, YOLO-based instrument detection, motion analysis, and a trainee dashboard to help make surgical assessment faster and more data-driven.

## Features

- Upload arthroscopy videos through a web interface
- Score performance across stability, efficiency, and precision
- Generate visual report images for each analysis run
- Store saved trainee sessions in a personal dashboard
- Support Google OAuth login for user accounts
- Train and reuse custom YOLOv8 weights for instrument tracking

## Setup

1. Create and activate a virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Copy the environment template and fill in your keys:

```bash
cp .env.example .env
```

4. Make sure a trained model exists at `weights/best.pt`.
   If you do not have custom weights yet, train them with `train_yolo.py` or update `analyzer.py` to point to a different checkpoint.

5. Run the app:

```bash
python app.py
```

6. Open `http://127.0.0.1:5000`.

## Environment Variables

- `ANTHROPIC_API_KEY`: API key for AI mentor feedback
- `GOOGLE_CLIENT_ID`: Google OAuth client ID
- `GOOGLE_CLIENT_SECRET`: Google OAuth client secret
- `SECRET_KEY`: Flask session secret

## Training Workflow

1. Extract frames from surgical videos with `extract_frames.py`.
2. Annotate/export the dataset into `dataset/`.
3. Train the detector with `train_yolo.py`.
4. The training script copies the best checkpoint to `weights/best.pt`.

## Notes

- Local datasets, trained runs, generated reports, SQLite files, and secrets are intentionally excluded from version control.
- The app currently expects model weights at `weights/best.pt`.
- Existing sample outputs can be regenerated by uploading videos through the web UI.
